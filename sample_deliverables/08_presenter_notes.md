# Presenter Notes

## Why this design fits the customer
- The interface feels operational, structured, and trustworthy, which suits a manufacturing environment
- The strong blue and dark navy create immediate brand ownership without sacrificing readability
- The teal accent adds modernity without making the interface feel overly consumer-like

## What the audience should notice first
- The platform feels branded from the first screen
- The communication model supports both leadership messaging and frontline operational updates
- The mobile experience looks intentional, not secondary

## What is handled through standard Staffbase branding
- Main branded color direction
- Navigation header expression
- Menu identity
- Accent color language
- Overall brand fit across core surfaces

## What may require Custom CSS refinement
- Bottom navigation selected-state styling
- Badge style polish
- Optional widget or localized content treatment
- Certain presentation-level nuances not covered by standard branding settings

## Suggested demo flow
1. Start with the branded home/news experience
2. Show how leadership communication, safety updates, and plant-level news coexist in one experience
3. Move to the mobile view and highlight bottom navigation and frontline usability
4. Explain the difference between standard branding configuration and controlled CSS refinement
5. Position the proposal as realistic, scalable, and brand-aligned

## Suggested talk track
- “This concept is designed to feel recognizably Northstar from the first interaction.”
- “We intentionally started with what can be achieved through standard Staffbase branding.”
- “Where we want a little more polish, we introduce a very small and controlled CSS layer.”
- “That gives us a strong balance between brand fit, rollout realism, and maintainability.”

## Likely customer reactions to prepare for
- “How much of this is configuration versus custom work?”
- “Will this hold up on mobile for frontline employees?”
- “Can local plants still have some room for tailored communication?”
- “How quickly could a branded pilot experience be prepared?”
